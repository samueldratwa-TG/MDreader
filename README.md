# MD/MDX Viewer — Comfortable MD File Viewer

A single, lightweight, and independent HTML file (~188 KB) designed to comfortably display and compare MarkDown (MD) and MDX files directly in your browser.
The tool [was created out of a simple need](https://sadot.click/blog/mdreader-story): to comfortably view these MD files without installing complex environments, setting up local servers, or uploading private files to the cloud. It can run directly from the web or work 100% offline.

---

## 🌟 Key Features

| Feature | Benefit |
| --- | --- |
| **Format Support** | Works seamlessly with `.md` and `.mdx`. For MDX, JSX tags are displayed as visual labels. |
| **Dual Pane Comparison** | View a single MD file or two files side-by-side. Perfect for comparing versions or translations. |
| **Smart Drag & Drop** | Simply drag MD files into the browser. You can drag two files together, and they will automatically open side-by-side. (If more than two are dragged, only the first two will be loaded). |
| **Direction Control (RTL/LTR)** | Each pane has an independent direction button for Hebrew MD files. |
| **Scroll Sync** | In dual-pane mode, you can align the pages so they match up face-to-face, and then click the "Scroll Sync" button (top right). From then on, both panes will scroll together in full sync. |
| **Reload** | Edited the original MD file externally? Click ↻ Reload and the changes reflect immediately. |
| **Useful Tools** | Word count, Syntax Highlighting, and Dark/Light mode. |

## 🚀 Quick Start Guide

The tool is very intuitive and ready for immediate action:

**Option A: Run Directly from Browser**
Visit the [Project's GitHub Pages](https://samueldratwa-TG.github.io/MDreader/) and start working.

**Option B: Offline Work (Download)**
1. **Download:** [Download the index.html file](https://github.com/samueldratwa-TG/MDreader/raw/master/index.html).
2. **Open:** Double-click the downloaded file.
3. **Loading MD Files:**
   * Drag an MD file into the window.
   * You can load or drag **two files together** — they will open instantly in both panes (side-by-side). If you select more than two, the system will only load the first two.
4. **Day-to-day Use & Sync:**
   * Change the pane ratio by dragging the divider (double-click to reset to 50/50).
   * **Scroll Sync:** Align the text in both panes so they are matched up, then click the "Scroll Sync" button located at the top right. You can now scroll both files together.
   * Use the RTL button for each side independently if needed.

# MD/MDX Viewer — כלי להצגה נוחה של קבצי MD

קובץ HTML בודד, קליל ועצמאי (~188 KB) שנועד להצגה נוחה ולהשוואת קבצי MarkDown (MD) ו-MDX ישירות בדפדפן שלכם.
הכלי [נוצר מתוך צורך פשוט](https://sadot.click/blog/mdreader-story): להציג בצורה נוחה את קבצי ה-MD האלה בלי להתקין סביבות עבודה, בלי להרים שרת מקומי ובלי להעלות קבצים פרטיים לענן. הוא יכול לרוץ ישירות מהרשת או לעבוד 100% אופליין.

| **תמיכה בפורמטים** | עובד חלק עם סיומות `.md` ו-`.mdx`. במקרה של MDX, תגיות JSX מוצגות כתוויות ויזואליות. |

| **ממשק Drag & Drop חכם** | פשוט גררו קבצי MD לתוך הדפדפן. ניתן לגרור שני קבצים יחד והם יפתחו אוטומטית זה לצד זה. (בגרירה של יותר משניים, רק השניים הראשונים ייטענו). |
| **שליטה בכיווניות (RTL/LTR)** | לכל חלונית יש כפתור כיווניות עצמאי עבור קבצי MD בעברית. |

| **רענון (Reload)** | ערכתם את קובץ ה-MD המקורי? לחצו על ↻ Reload והשינויים ישתקפו מיד. |

## 🚀 מדריך שימוש מהיר

הכלי מאוד אינטואיטיבי ומוכן לפעולה מיד:

**אופציה א': הפעלה ישירה מהדפדפן**
היכנסו ל-[GitHub Pages של הפרויקט](https://samueldratwa-TG.github.io/MDreader/) והתחילו לעבוד.

**אופציה ב': עבודה אופליין (הורדה)**
1. **הורדה:** [הורידו את קובץ ה-index.html](https://github.com/samueldratwa-TG/MDreader/raw/master/index.html).
2. **פתיחה:** לחצו פעמיים על הקובץ שירד.
3. **טעינת קבצי MD:**
   * גררו קובץ MD לחלון.
   * ניתן לטעון או לגרור **שני קבצים יחד** — הם יפתחו מיד בשתי החלוניות (צד-בצד). אם תבחרו יותר משניים, המערכת תטען רק את השניים הראשונים.
4. **עבודה שוטפת בסנכרון:**
   * שנו את יחס הגדלים על ידי גרירת המחיצה (לחיצה כפולה מאפסת ל-50/50).
   * **סנכרון גלילה (Scroll Sync):** יישרו את הטקסט בשתי החלוניות כך שיהיה מקביל אחד לשני, ולחצו על כפתור "Scroll Sync" הממוקם מצד ימין למעלה. כעת תוכלו לגלול את שני הקבצים יחד.
   * השתמשו בכפתור ה-RTL לכל צד בנפרד במידת הצורך.
